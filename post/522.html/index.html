我是周良粥凉，这是我所使用的利器

2015-08-02 应用 ， 工具 ， 杂记

介绍一下你自己和所做的工作。

我叫周良，95 年的熊孩子一枚。没读过大学，文化水平挺低的；喜欢折腾，极端的应用控。现在在一家不知名的创业公司干着 UI 设计师的工作，也在自我完善产品观，转型产品经理中。喜欢随手写点东西，偶尔也吐吐槽。

你都在使用哪些硬件？

iPhone 6（128G 深空灰）：如果说设计是我的兴趣，那么 iPhone 应该是让我热爱 UI 设计的根本原因。选择 128G 是因为上一台 iPhone5 不够我装应用了。而深空灰，是我认为在外观上相较于其它两款所能接受的。

13''MacBook Pro：12年款的，还带着光驱。由于经济原因，一直没有更换，大部分的作品都在这上面产出。对我来说还是够用的，但在运行一些大型软件时，会比较缓慢，甚至无法使用，不过现在已经开始工作了，希望可以攒够一台新的 Mac。

Apple Magic Mouse：不用离手即可快速切换不同桌面，在滚动画布（PS/Sketch）的时候也非常方便。

Kindle Paperwhite：在学校，很多个夜晚是它陪伴我度过的。

Ziiiro Celeste：说白了就是一块手表。戴手表有两个目的，一是获知时间，二是提醒自己做好时间把控。

软件呢？

OS X 上常用软件：

Sketch：常用的设计软件，2014 年中最正确的决定就是使用 Sketch3 进行设计，目前已经很少打开 Phtoshop 了。配合 Sketch Toolbox 安装各类插件，能更好的使用它。

Pixate + Framer Studio：动效与原型设计的软件实在太多了,尝试许多方案后，我使用 Pixate 完成一部分简单的动效，Framer Studio 完成相对较复杂，或者需要提供代码给开发进行参考的效果。

InVision：也是我常用的可交互原型制作工具。搭配使用 InVision Sync，可以将 Sketch 的设计稿同步到 InVision，很方便。

IconJar：Icon 素材管理工具，快速找到需要使用的图标。支持 SVG、PNG、Gif 三种格式。

RightFont：非常好用的字体管理工具，已与 PhotoShop、Sketch 两大设计应用集成。

Ember / Inboard：图片素材管理工具，在 iOS 上频繁使用 Ember for iOS，存储我的各类 APP 截图，通过 Dropbox 同步。后者是近半年开始使用，主要用于同步 Dribbble 上 like 的作品和保存收集的设计作品截图。

Paste：粘贴板管理应用，可以将所有复制过的内容（图片、文本、文件等）记录下来，并且一键快速调用。使用它之后，我在资料整理和绘制设计稿这些事上，效率提升了不少。

Moom：窗口布局软件，总会用到。

MinNode / XMind：思维导图工具，我更喜欢前者，在 iOS 上也经常用到。

Ulysses：在 Mac 上常用它写东西，现在你看到的这些文字，都是用它事先撰写的。

xScope：尺寸测量工具，经常会用到。

Alfred：快速打开应用，也使用了不少自定义功能。

Alternote：轻量级 Evernote 应用，支持 Markdown。

Dropbox：稳定的云同步功能，所有设计稿都存这上面。

Droplr：快速分享文件，生成短链接。

Wunderlist：GTD 应用。使用过很久的 OmniFocus，太重了，并不适合我。最终选择了它。美中不足的是没有清单归档功能，删除后就再也找不到了。

1Password：所有的密码都存在这里。

Pinbox：轻量级 Pinboard 客户端，国人开发作品。

PopClip：划词选项应用，有丰富的扩展插件库。

TextExpander：快捷输入神器。

Pushbullet：双向文件传输工具。

iOS 上常用软件：

Calendars 5：最常用的日历应用，在 iPhone 上提供了月视图，能够直观地查看当月事项的分布情况。

Tweetbot：最好用的第三方 twitter 客户端。

墨客：最好用的第三方微博客户端，配合分享功能上的「分享到印象笔记」来完成微博内容的稍后阅读和保存。

Drafts：先把文字写下来，再考虑这些内容可以放到哪里去，在 iOS 上大部分文字靠它输出。

Castro：听 Podcast，我喜欢它的交互。

Reeder / Unread：常用的两款 Rss 阅读应用，现在主要使用 Unread，排版非常舒服。Reeder 在文件夹分类上做的相对较好。

Mailbox：邮件应用，纯手势操作。Inbox 也是我常用的邮件客户端，但它的分类功能有时也会不靠谱，推送通知也几乎不可使用（我在 Mac 上没有找到合适的邮件客户端，所以一直在使用 Gmail 网页版）。

Sketch Mirror：在手机上预览 Sketch 的设计稿。

网易有钱：过去使用 DailyCost 和  MoneyWiz 2 ，前者专注于流水记账，后者管理各个资金账户，但太繁琐。最终选择了网易有钱。两者的结合体。并且，颜值也很高。

Screeny / Purrge：前者快速删除 iOS 上的截图; 后者可以快速多选需要删除的图片。

Day One：最好用的日记应用，习惯用手机写日记。

Pinswift：个人认为是最好的 Pinboard iOS 客户端。

iA Writher Pro：iPhone 上常用的 Markdown 编辑应用，长篇的内容都是用它写的。

格志日记：晨间日记/问答日记。起床、睡觉前都会用它记录一些东西，一段时间下来，我可以感觉到我有明显的改变。

好奇心日报：内容暂且不说，光是使用这款应用，就让我很舒服。

丁香医生 / 用药助手：生活中经常会生一些小毛病，这事不能马虎。

WALKUP：目前为止见过颜值最高，也是最好玩的计步 APP。

即刻：基于兴趣的极简信息推送工具。

Surge：iOS 上的 web 开发、代理工具。可以配合 Shadowsocks 进行翻墙。

另外常用的还有 虾米音乐、网易云音乐、Memoir、Simplenote、Spectrum 等，前面写过一篇有关 iOS 上提升效率的应用推荐文章，其中大部分我一直都在使用，如果你有兴趣，可以点击「这里」阅读。

你平时获得工作灵感的方式有哪些？

与人聊天：除了锻炼我的沟通能力外，还能获取许多的信息，也可以给自己的知识体系做一个整理。

Podcast：常听的有 比特新声、IT公论、内核恐慌、流行通信、狗熊有话说 等。

NEXT：发现一些新的产品，最爱的是「产品集」栏目。近期上线的「Maker」，另外 NEXT 在一些体验和细节上做的很棒。

Mindstore.io：我喜欢他的「MindTalk 线场」，可以看到他人对某一款产品的见解。

看：常在 Dribbble / Pinterest / Behance 上面浏览、收藏各类作品。看的东西非常杂乱，并且提升审美这事需要日积月累。

阅读：Medium、简书、Rss 订阅、恐怖悬疑小说、各种杂乱的书籍报刊。

写东西：主要目的是让自己静下来。

把玩各类 APP：看周围朋友手机上都装了什么应用。寻找各类优秀的应用，每个人对优秀的定义都不同，在我眼里「设计精美」或者「满足实际场景应用」即为优秀。利器也是一个不错的途径，我喜欢这类开放式分享的内容。目前手机上在使用的迷雾世界就是通过利器发现的，类似的还有主屏（一个微信公众）。这里有我的一篇文章，分享了我发现优秀 iOS 应用的途径（写于一年前，如果你有更好的，欢迎告诉我）。

你最理想的工作环境。

一个不会给人造成压抑感的空间，有大木桌、新鲜的空气、一张带有背部支撑的椅子，网速要快。如果我的显示器后面是一堵墙，那么更加完美（当然，这是一个小玩笑）。

推荐一件生活中的利器给大家。

晨间日记 / 睡前日记：日记是一个审视自己的过程，晨间日记与问答日记是一种很适合做回顾和规划的方式。坚持使用一段时间后，可以明显感觉到自己的变化。
